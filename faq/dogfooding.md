# 비즈스프링이 자사 제품을 직접 사용하나요? (도그푸딩)

## 도그푸딩이란?

도그푸딩 (dogfooding) — "자기 회사 개밥을 자기가 먹는다(eat your own dog food)"라는 영어 표현에서 유래. **자사가 만든 제품을 자사가 먼저 직접 사용하여 효과·문제점을 검증하는 운영 방식**입니다.

## 비즈스프링의 도그푸딩

비즈스프링은 자사가 개발한 **PREMIUM_GEO** (Generative Engine Optimization) 엔진을 **자사 Wiki에 가장 먼저 적용**하고 있습니다. 본 사이트 wiki.bizspring.co.kr이 그 결과물입니다.

### 적용 범위

| 자산 | 적용 |
|----|----|
| Schema.org JSON-LD | 33페이지 전체 |
| Trust 7묶음 (E-E-A-T 강화) | 33페이지 전체 |
| BreadcrumbList 계층 | 33페이지 전체 |
| Service.provider / Article.publisher | 의무 채움 |
| sameAs (Knowledge Graph 연결) | LinkedIn / 메인 사이트 등 |

### 자사 솔루션 사용률 75%

비즈스프링은 자사가 운영하는 SaaS 8종 중 6종을 사내에서 직접 사용:

- ✅ **LIFT** (UTM 표준화 + 자동 매칭) — 4월 세션 1,305건
- ✅ **AIR** (광고비 통합 리포트) — 4월 광고비 391,983원
- ✅ **GEOcare** (GEO 측정 풀)
- ✅ **mDP** (데이터 마트)
- ✅ **LOGGER** (이벤트 로그)
- ✅ **MA** (Marketing Automation)
- ❌ GA4 SaaS (외부 사용)
- ❌ CRM

### 4월 baseline (실측)

| 항목 | 값 |
|----|----|
| LIFT 세션 | 1,305 |
| LIFT 매출 | 104,701원 |
| 광고비 (AIR 통합) | 391,983원 |
| **AI 검색 인용 (LLM 클릭)** | **22건** (OpenAI 11 / Gemini 8 / Perplexity 3) |
| GEO 점수 (메인) | 68점 |
| EEAT TOP 페이지 | 76점 |
| SoM (Share of Mention) | 26.30% |

## 왜 도그푸딩하는가?

### 1. 영업 무기

자사 데이터 = 영업 케이스 1건. **"우리도 자사에서 적용하고 있습니다"** = 가장 강력한 영업 메시지.

### 2. 사용성 검증

자사 운영 중 발견한 사용성 이슈 = 즉시 본방 PREMIUM_GEO에 피드백 → 제품 개선.

### 3. Citation Moat 자체 구축

W5 (2026-05-19) 발행 시 33페이지 평균 Trust 60+ 도달 → AI 검색에서 "비즈스프링"이 권위 있게 인용되는 인용 해자(Citation Moat) 자체 구축.
